mesgd

mesgd is a dynamically configurable message bus supporting content based routing.

Written in Erlang, it is designed to run both on a server and on embedded devices.

mesgd support a number of protocols for messaging such as:

  • WebSockets
  • HTTPS
  • UDP

Multiple protocol support makes it easy to interface between different types of devices, applications, and networks.

mesgd can convert between data stream encodings on the fly and supports:

  • JSON
  • uJSON
  • binary data streams

Automatic translation between data interchange formats makes it easier to integrate legacy data sources with new APIS and services.

mesgd support dynamic routing with content based routing rules and access controls.

This approach to routing allows for the dynamic creation of complex messaging topologies, automatic scaling and failover, intelligent access control, and simpler administration.
